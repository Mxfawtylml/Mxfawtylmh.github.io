Github Navigator

USACO

Access repository here

The United States of America Programming Olympiad is a programming competition that serves as qualification for the USA IOI team. It is held four times every year and is my primary focus in competitive programming and where you will find the majority of my algorithmic code. Most of my USACO code will be written in java.

CCC

Access repository here

The Canadian Computing Competition is a programming competition that serves as qualification for the Canadian Computing Olympiad. Unlike USACO, it is held annually in february. Thus, you will find significantly less code in this directory, albeit the questions are equally as challenging. Most of my CCC code is written in java, with an exceptions of python for earlier contests.
